# SilksongAccess-Dev
Development of the Hollow Knight: Silksong BVI accessibility mod

# Current features:
- Accessible interact prompts, item pickups, and collectable pickups
- Accessible shop UI
- Accessible inventory UI
- Most menus are accessible on controller
- NPC dialogue
- Simple exploration announcements
- Healing recieved and damage taken
- Spending and gaining silk
- Defeating enemies
- Cutscenes starting and ending
- Prompt to skip cutscenes

# WORKING ON:
- Full exploration and navigation
- Some parts of the inventory and map menus need more work
- Some on screen text that appears during gameplay
- Audio description for cutscenes
